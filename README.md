# 📂 Project Planner

> A lightweight, highly responsive project and task management application designed for smooth workflow management and real-time state tracking.

[![Live App](https://img.shields.io/badge/Live_App-2563EB?style=for-the-badge)](https://arian-planner.darkube.ir)

---

## 🚀 Key Features

* **Dynamic Screen Routing:** Clean UI transitions between project creation, detailed project views, and contextual fallbacks without unnecessary re-renders.
* **Task Management:** Real-time sub-task creation and deletion isolated to individual project scopes.
* **Persistent Local Storage:** Automatic synchronization with `localStorage` so user data persists across browser sessions.
* **Modern Component Architecture:** Built using React functional components, custom hooks (`useState`, `useEffect`, `useRef`), and styled with **Tailwind CSS** for responsive design.
* **Containerized Deployment:** Packaged with **Docker** and continuously deployed on Darkube PaaS.

---

## 🛠️ Tech Stack

* **Frontend:** React, HTML5, Tailwind CSS
* **State Management:** React Hooks & Local Storage API
* **DevOps & Hosting:** Docker & Hamravesh
