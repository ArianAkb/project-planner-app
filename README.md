# 📂 Project Planner

A lightweight, highly responsive project and task management application built to practice modern React concepts. 

### 🚀 Features
* **Dynamic Views:** Easily toggle between project creation, project details, and a fallback screen.
* **Task Management:** Create and delete sub-tasks tied directly to specific projects.
* **Persistent Memory:** Powered by `localStorage` so your data survives a browser refresh.
* **Modern React:** Built using React Hooks (`useState`, `useEffect`, `useRef`) and styled with Tailwind CSS.
